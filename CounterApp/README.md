# Counter App

## About the Project:

A simple JavaScript counter application that increases, decreases, and resets a number using button clicks.
Built to practice DOM manipulation and event handling.

## Project Flow:

1. User opens the app.
2. Counter starts at 0.
3. Clicking Increment increases count by 1.
4. Clicking Decrement decreases count by 1.
5. Clicking Reset sets count back to 0.
6. UI updates instantly using DOM manipulation.
