# Calculator

## About The Project:
It is a calculator project that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. It provides an interactive interface where users can enter numbers and calculate results instantly.it is done using basic HTML,CSS and JavaScript.

## Flow:
- User clicks number buttons to enter values
- User selects an operator (+, -, *, ÷, %)
- Input is displayed on screen
- When user clicks "=":
  - JavaScript evaluates the expression
  - Result is displayed
- User can clear input using AC button
