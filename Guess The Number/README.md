# Guess The Number Game

## About The Project:
It is a beginner-friendly JavaScript project where the user tries to guess a randomly generated number. The project uses DOM manipulation, event listeners, conditionals, and random number generation to create an interactive game.

## Flow:
- Generates a random number within a given range
- Takes user input and checks the guess
- Gives feedback like:
  - Too High 
  - Too Low 
  - Correct
