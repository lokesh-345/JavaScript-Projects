# To-Do List

## About The Project:
This is a beginner-friendly JavaScript project that helps users manage daily tasks by adding, completing, and deleting tasks. It provides an interactive way to organize work and improve productivity using DOM manipulation and event handling.

## Flow:
- User enters a task in the input field
- Clicks Add button (or presses Enter)
- JavaScript reads the input value
- A new task item is dynamically created and added to the list
- User can:
  - Mark task as completed 
  - Delete task 
- UI updates instantly without refreshing the page
