# Rock Paper Scissor Game

## About The Project:
A simple interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript where the user plays against the computer.

## Flow:
1. User clicks Rock, Paper, or Scissors
2. JavaScript captures the click using event listeners
3. Computer randomly selects one option using Math.random() function
4. Both choices are compared using game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
5. Result and score are updated on screen
